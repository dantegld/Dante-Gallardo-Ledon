Hamburger Menus

Voorbeelden:


Stap 1

Download de hamburger.css file van https://jonsuh.com/hamburgers/.

Stap 2

Link hamburger.css aan je project.

<link href="libraries/hamburgers.css" rel="stylesheet">

Stap 3

Voeg de hamburger class toe aan je HTML elementen:

<div class="hamburger hamburger--mode">
<div class="hamburger-box">
<div class="hamburger-inner"></div>
</div>
</div>
