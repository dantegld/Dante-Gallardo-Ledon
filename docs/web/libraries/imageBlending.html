Image Blending


Stap 1

Download de granim.js file van https://sarcadass.github.io/granim.js/.

Stap 2

Link granim.js aan je project.

<script src="libraries/granim.js"></script>

Stap 3

Voeg de volgende code toe aan je script.js:

var granimInstance = new Granim({ element: '#canvas-image-blending', direction: 'top-bottom', isPausedWhenNotInView: true, image : { source: 'img/bg1.jpg', blendingMode: 'multiply' }, states : { "default-state": { gradients: [ ['#29323c', '#485563'], ['#FF6B6B', '#556270'], ['#80d3fe', '#7ea0c4'], ['#f0ab51', '#eceba3'] ], transitionSpeed: 7000 } } });
